# LOVE CHINESE 爱中文

Production foundation for `lovechinese520.com`, built with Next.js App Router, TypeScript, Tailwind CSS, responsive navigation, SEO metadata, sitemap, and robots support.

## Run Locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
